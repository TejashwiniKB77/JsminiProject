JavaScript Mini Project

This project is a responsive college website enhanced using JavaScript (ES6+) to add interactivity, dynamic content handling, and browser-based functionality.
The project is built on top of existing HTML and CSS with all logic running completely in the browser (no backend).

Technologies Used HTML5 CSS3 JavaScript (ES6+) Browser Web APIs
